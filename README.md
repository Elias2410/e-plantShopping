## 🌿 E-Plant Shopping App

## 📌 Description
E-Plant Shopping is a front-end web application that allows users to browse plant products and add them to a shopping cart.

This project demonstrates state management using Redux and a modern React setup powered by Vite.

## 🚀 Features
 - 🌱 View a list of plant products
 - 🛒 Add items to cart
 - 🔄 Update cart items dynamically
 - ⚡ Fast performance with Vite
 - 🎯 Clean component-based structure

## 🛠️ Technologies Used
 - React (with JSX)
 - Redux (state management)
 - JavaScript (ES6+)
 - CSS
 - Vite
